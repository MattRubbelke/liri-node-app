# liri-node-app
This is the Language Interpretation and Recognition Interface! Find your favorite artists' upcoming concerts, spotify songs, and search movies. 

# How to Use
In your command terminal, enter the following syntax: "node liri.js <<command>> parameter" (ex: node liri.js concert-this lady gaga).
The following commands may be used:
* concert-this: Search for concerts;
* spotify-this: Search for songs that are available on Spotify;
* movie-this: Search for movies; and
* do-what-it-says: Reads the random.txt file, and spotifies the song on the random.txt file. No parameter is needed.

For parameters, there is no specific syntax needed. There is no restriction on the maximum number of words used. Ensure you spell your search terms correctly.

# What is needed to use the app
You will need to download the [Node-Spotify-API] API (https://www.npmjs.com/package/node-spotify-api), the [Axios](https://www.npmjs.com/package/axios), the [Moment](https://www.npmjs.com/package/moment), and the [DotEnv](https://www.npmjs.com/package/dotenv) packages.
